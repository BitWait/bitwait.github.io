## Algorithms
### 题意：
- 给定一个二叉树，原地将它展开为一个单链表。
### 思考：
- 将树的左子树与右子树都转化成链表，然后，将右子树接在左子树的后边
- 采用后序遍历的方式，从左向右，从低向高处理
### 代码实现：
```
class Solution {
public:
    void flatten(TreeNode* root) {
        if(!root)
            return ;
        flatten(root->left);
        flatten(root->right);

        TreeNode* tmp = root->right;
        root->right  = root->left;
        root->left = nullptr;

        while(root->right)
            root = root->right;
        root->right = tmp;
    }
};
```
